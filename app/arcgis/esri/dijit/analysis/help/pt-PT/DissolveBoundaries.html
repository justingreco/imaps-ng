Dissolver Limites

Áreas que se sobrepõem ou partilham um limite são fundidas de forma a formar uma única área.

Pode controlar quais os limites a serem fundidos, especificando um campoPor exemplo, se tem uma camada de municípios, e cada município tem um atributo State_Name , pode dissolver os limites utilizando o atributo State_Name. Se os municípios adjacentes tiverem o mesmo valor para State_Name serão fundidos. O resultado final é uma camada com todos os limites fundidos.

Escolher método Dissolver

Pode escolher um dos dois métodos para fundir a área dos elementos:

  • Áreas que se sobrepoem ou são adjacentes—Duas ou mais áreas cujos limites sobrepõem uns aos outros serão fundidas numa área. Além disso, as áreas que são adjacentes (isto é, que partilham um limite) serão fundidos apenas numa área.
  • Áreas com o mesmo campo de valor—Áreas que se sobrepoem ou partilham uma fronteira comum serão fundidas numa área se tiverem o mesmo valor do campo.

    Por exemplo, se tem uma camada de paises, e cada pais tem um campo State_Name , pode disolver as fronteiras usando o campo State_Name . Se os municípios adjacentes tiverem o mesmo valor para State_Name serão fundidos. O resultado final é uma camada com todos os limites fundidos.

Se Use uma extensão de mapa actual está validado, apenas elementos na camada de entrada visível dentro da extensão de mapa actual serão dissolvido. Se não estiver validado, todos os elementos na camada de entrada serão disolvidas, mesmo que estejam fora da extensão de mapa actual.

Adicionar estatística

Quando as áreas são fundidas, é possível resumir os respectivos atributos. Por exemplo, se está a disolver as fronteiras entre municipios baseado no State_Name, e cada condado tem um campo População , pode calcular a soma de População para o estado.

Tipos de estatística disponíveis:

  • Soma—Adiciona o valor total para o campo especificado
  • Média—Calcula a média para o campo especificado
  • Minímo—Localiza o menor valor para todos os registros do campo especificado
  • Máximo—Encontra o valor maior para todos os registros do campo especificado
  • Desvio -padrão-Encontra o desvio-padrão dos valores do campo especificado

Nome da camada resultante

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que forneça outro nome.

Ao utilizar a caixa suspensa Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
